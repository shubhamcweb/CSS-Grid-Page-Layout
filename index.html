🖥 CSS Grid Webpage

Hello there! This is my first try on using CSS Grid for a responsive webpage layout. I have also used CSS Grid for the little picture collage that you see below and there's also some CSS Flexbox sprinkled over here and there! Yep. They work great together!

This is also the first time I deployed a collapsible Navigation bar functionality using JavaScript to enhance the user experience on mobile interfaces.

Overall, this was an incredibly exciting project, and I got to learn a ton! Special thanks to Zach Gollwitzer for his amazing tutorials on YouTube, without which the web development journey would be so much difficult.
